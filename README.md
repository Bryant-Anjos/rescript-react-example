# ReScript + React Example

Small web app built to study **ReScript** integrated with **React**. Covers ReScript syntax, JSX bindings, component composition, and how to interop with JavaScript.

## Tech Stack

- **ReScript** — statically typed functional language that compiles to JavaScript
- **React** — UI library
- **Vite** (via ReScript toolchain)

## Getting Started

```bash
npm install
npm run start   # build and watch
```

Open `src/Demo.bs.js` to see the compiled output.

## What This Explores

- ReScript module system and type inference
- React component definition in ReScript
- JSX support in `.res` files
- Zero-cost interop with JavaScript
